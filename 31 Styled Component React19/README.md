# React Toggle Cards with Styled Components

This is a simple React application that demonstrates the use of **Styled Components** to create styled buttons and cards. The application includes a **toggle button** that shows or hides a list of cards dynamically.

##  Technologies Used

- React (v18+)
- Styled Components


##  Features

- Toggle button to show/hide cards
- Styled using `styled-components`
- Smooth scaling effect on button click
- Clean and modular structure
